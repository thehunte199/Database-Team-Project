HELP

Deleting Information

To delete information from the database, please complete the following steps:
  1. Navigate to the delete page.
  2. Select the table you wish to delete from.
  3. Enter the primary key of the entry you wish to delete
  4. Submit your request

Inserting Information

To insert information into the database, please complete the following steps:
  1. Navigate to the insert page.
  2. Select the tab that corrosponds to the table you wish to insert into.
  3. Fill out the required information.
  4. Submit your request

Updating Information

To update information in the database, please complete the following steps:
  1. Navigate to the update page.
  2. Select the table you wish to update information in from the corrosponding drop down menu.
  3. Identify the entry you wish to modify by selecting its primary key.
  4. Choose the attribute that you wish to modify.
  5. Enter in a new value for the attribute. The new value can be empty or non-empty.
  6. Submit your request.

View the Database

To view the contents of the database, please complete the following steps:
  1. Navigate to the view page.
  2. Select the table you wish to view.
  3. Submit your request.
    4. The information displayed can be sorted based on different attributes, in ascending and descending order.

Viewing Derived Information

Derived information are a series of miscellaneous queries that answer "I would like to know..." questions. To view these queries, please complete the following steps:
  1. Navigate to the derived page.
  2. Select the question from the drop down menu that corrosponds to what you want to know.
  3. Submit your request
