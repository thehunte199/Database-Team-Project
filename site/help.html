HELP

Deleting Information

To delete information from the database, please complete the following steps:
  1. Click on the Delete button located on the left to navigate to the delete page.
  2. Select the table you wish to delete from.
  3. The attributes of the table should be auto populated.
  4. Select the condition operation you would like to use (eg. =, >, != …)
  5. Type the value of the row that you would like to delete
  6. Submit your request by pressing the Delete button below the text fields.

Inserting Information

To insert information into the database, please complete the following steps:
  1. Click on the Insert button located on the left to navigate to the insert page.
  2. Select the tab that corresponds to the table you wish to insert into.
  3. Fill out the required information with the red asterisk(*).
  4. Enter any other information you would like to add.
  5. Submit your request by pressing the Insert button below the text fields.

Updating Information

To update information in the database, please complete the following steps:
  1. Click on the Update button located to the left to navigate to the update page.
  2. Select the table you wish to update information in from the corresponding drop down menu.
  3. Choose the attribute that you wish to modify.
  4. Enter in a new value for the attribute.
  5. In the ‘Where’ section enter an attribute of the table (eg. For member use bannerId = ‘900______’).
  6. Submit your request by pressing the Update button below the text fields.

View the Database

To view the contents of the database, please complete the following steps:
  1. Click on the View button located to the left to navigate to the view page.
  2. Select the table you wish to view.
  3. Choose an attribute of the table to sort on
  4. Choose which ordering to sort Ascending or Descending
  5. Submit your request by pressing the View button below the text fields. The information displayed can be sorted based on different attributes, in ascending and descending order.

Viewing Derived Information

Derived information are a series of miscellaneous queries that answer "I would like to know..." questions. To view these queries, please complete the following steps:
  1. Navigate to the derived page.
  2. Select the question from the drop down menu that corrosponds to what you want to know.
